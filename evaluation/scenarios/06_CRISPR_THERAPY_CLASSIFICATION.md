# 시나리오 6 — CRISPR 신약 정부 분류·펀딩 매핑

> **질문**: CRISPR 유전자 편집으로 희귀질환 치료 사업을 시작하려고 합니다. 우리 연구가 어떤 정부 분류 코드에 들어가는지, 어떤 펀딩 사업이 가장 적합한지, 관련 과제·논문은 어떤 것이 있는지 알려주세요.

## 🎯 분석 요약

CRISPR 유전자 편집 치료제 분야는 표준 분류 추천 시 입력 텍스트 정제가 필요하지만, 정부 R&D 분류 체계에서는 **생명(LA 생명과학, LC 보건의료)** 대분류 산하 **신약·의약품개발(LC03) + 분자세포생물학(LA01)** 결합 트랙이 가장 적합합니다. NTIS 표준 분류 22개 대분류 중 **생명(2), 인공물(03 ED 전기·전자, EE 정보·통신), 자연(01 NA·NB·NC)** 4축 매핑이 가능하며, 부처는 **보건복지부 + 과학기술정보통신부** 다부처 트랙이 표준입니다. 학술 DB DDC(듀이 십진 분류) 검색은 권한 신청 중이며, 본 사업 진입 시 글로벌 학술 인덱스 매핑 가능합니다.

### 학술 DB DDC 검색 (학술·특허 DB)

- 권한 미부여 (E4302) → 권한 신청 중. 권한 확보 시 듀이 십진 분류 기반 CRISPR 관련 글로벌 학술 인덱스 매핑 가능.

## 🔍 심층 분석

1. **표준 분류 추천 실패 진단**
   - 입력 텍스트 약 350자(요건 450바이트 이상) — **분류 추천 API는 학술적 키워드·기술 설명 충분치 않다고 응답 (-2002)**.
   - 본 사업 기획서 진입 시 500~800자로 보강하여 재호출 필요.
   - 권장 키워드: "CRISPR-Cas9 표적 특이성, 단염기 편집, AAV 벡터, 면역 회피, siRNA 전달 시스템, GMP 생산 공정, 동물 모델 효능 평가, IND 신청, FDA·EMA 인허가, 임상 1상".

2. **대표 부처·사업 매핑 (LC03 + LA01 + NC 결합)**
   - **보건복지부**: 신약·의약품개발 트랙 (LC03)
   - **과학기술정보통신부**: 바이오·의료기술개발, 분자세포생물학 (LC + LA)
   - **다부처 사업**: 인공지능 활용 혁신신약 발굴(시나리오 1 참조) — CRISPR도 적합

3. **부처 매칭 우선순위**
   - 🥇 보건복지부 — 임상 직결 신약 트랙 (단일 과제 매칭 5~50억 원)
   - 🥈 과기정통부 — 기초 연구 (단일 매칭 2~10억 원)
   - 🥉 식약처 P60 — 의료 안전성 평가 트랙 보조

4. **국제 분류 호환성**
   - 한국 NTIS001 표준 분류 ↔ DDC ↔ ICD-10/11 ↔ MeSH 매핑이 필요. 학술 DB DDC 검색 권한 확보가 핵심.
   - CRISPR 관련 글로벌 DDC: 572 (Biochemistry), 579 (Microorganisms), 615 (Pharmacology) 후보.

5. **희귀질환 분류 트랙**
   - **희귀·난치성 질환** — 보건복지부 별도 트랙 존재 (KCD 분류 D01~D89, E70~E90 등).
   - 정책 명분 강함 → 매칭률·예산 우대 가능.

## 💼 비즈니스 권고

**단기 (6개월~1년)**
- 사업 기획서 500자 이상 상세 작성 후 표준 분류 추천 재호출 (정확한 매핑 코드 확보).
- 보건복지부 신약·의약품개발 트랙 사업 공고 일정 점검 (한국보건산업진흥원).
- 학술 DB DDC 검색 권한 확보 진행.

**중기 (1~2년)**
- 다부처 사업(과기정통부+복지부) 단일 과제 5~15억 원 매칭 확보.
- 표적 희귀질환 1건 선정 — 베타지중해빈혈, 척수성근위축증(SMA), 헌팅턴병 등 단일 유전자 변이 질환.
- IP — CRISPR + AAV 벡터 + 표적 가이드 RNA 1건 PCT 출원.

**장기 (3년+)**
- 임상 1상 IND 신청 (식약처) → 50~100억 원 정부 매칭 + 자체 자금.
- 글로벌 라이선스 또는 합자 (Editas, Intellia, CRISPR Therapeutics).

## ⚠️ 리스크·고려사항

1. **분류 추천 한계** — 표준 분류 API는 텍스트 길이 요건 엄격. 일관성 위해 표준 사업기획서 양식 활용.

3. **윤리·규제** — 생식세포 편집 금지, 체세포 편집 가이드라인 진행 중. 식약처 가이드 모니터링.
